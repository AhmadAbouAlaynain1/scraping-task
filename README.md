# Scraping Program

#### How to Run 1 Level Spider

1. Clone the repo
2. ```
   pip install -r requirements.txt
   cd scrapy_task
   scrapy crawl oneLevel -O companies_level_one.json
   ```

#### How to Run 2 Level Spider

1. Clone the repo
2. ```
   pip install -r requirements.txt
   cd scrapy_task
   scrapy crawl twoLevel -O companies_level_two.json
   ```
